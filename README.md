# 🎲 Knockout Dice Game

## About

This is a browser-based dice game built with JavaScript.
Players roll dice and try to achieve specific combinations to win.

## Features

* Random dice rolling
* Game logic for scoring
* Interactive UI updates
* Simple game rules

## Technologies

* HTML
* CSS
* JavaScript

## How to run

1. Clone the repository
2. Open `Knockout.html` in your browser

## How to play

* Click "Roll Dice"
* Try to reach the target score
* Follow the game rules displayed on screen
